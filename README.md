# TaskDragger

A simple task management app inspired by **Trello**, built with **Godot Engine**.  

💡 **Note:** This application works entirely **offline** — no internet connection is required.

---

## Features

- Create and organize **boards**, **lists**, and **cards**  
- Drag & drop cards between lists  
- Edit card titles and colors  
- Right-click to delete or add cards
- Search features for cards
- Filter by "completion" and colors
- Settings menu with : Automatic theme switch based on the OS, column width and spacing
- 9 colours of board are available
- Path to saves can be switch at any moment (If save.json is created manually it will be detected automatically)
---

## Controls

- Long click & drag to move cards or columns   (There is also an auto scroll vertical and horizontal during the drag)
- Click to edit card/column  
- Right-click for card options
- Use enter to quicly make multiple cards or columns
- Use Alt+Enter to create a newline
